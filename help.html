Description about the Task Management App.

How works Add Task?

  • TITLE: You can here introduce the Name of the Task.
  • URGENCY: Help to set the urgency status from the task either high urgent or low urgent.
  • IMPORTANCE: This area is used to place the priority of a specified task,similary to urgency high or low.
  • DATE: To create a current date, you click on the square item and select the desired date.
  • DESCRIPTION: In this field you can describe your task.
  • CANCEL: Button to abandon a task.
  • ADD TASK: Button to save a task.

How are the tasks displayed in Matrix?

In this area all Tasks you created are organized in four Categories that help you to prioritise them better.They are displayed according to importance and urgency.

  • ADD TASK Link: Click on the add Task Link to create a new Task.
  • MATRIX Link: Press on the Matrix Link to view all Tasks categorized by priorities.
  • LIST Link: Click on the List Link to view the listed Tasks.
  • DELETE Task: Click on the delete button to delete a specified task.
  • SCROLL: Scroll the Button to see all Tasks displayed in a list Form.

How are the Tasks displayed in the List?

In this section the created Tasks are saved as a List.

  • DATE: The date when the Task was created.
  • NAME: Represents the Name of the Task.
  • PRIORITY STATUS: Communicate the level of a specified Task: low, medium or high.
  • DESCRIPTION: Represents the Description from the Task.
  • DELETE TASK: Delete a created Task.
